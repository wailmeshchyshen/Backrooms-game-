# The Backrooms — SCP Descent

A browser-based first-person horror maze. Descend through **10 procedurally generated Backrooms levels**, each stalked by a different **SCP anomaly**. Find the buzzing green exit before it finds you.

Built as a single self-contained HTML file with **Three.js** — runs on desktop and mobile, and installs as an offline **PWA**.

## Gameplay

Every level is a fresh procedural maze with its own anomaly and survival rules: keep **SCP-173** in your sight or it moves when you blink, never look at **SCP-096**, walk (don't sprint) past the sound-hunting **SCP-939**, and there's no hiding from **SCP-106**, who phases through walls. Manage sanity, stamina, and almond water while navigating by flashlight and a reveal-as-you-go minimap.

## Features

- 10 levels, 10 unique SCP behaviors
- Procedural mazes — no two runs alike
- Dynamic shadows, fog, film grain, ambient audio
- Sanity / stamina / collectible systems
- PC (WASD + mouse) and mobile (touch) controls
- Installable PWA, playable offline

## Anomalies

SCP-173 · 096 · 049 · 939 · 106 · 966 · 035 · 457 · 682 · 1471

## Controls

`WASD`/arrows move · mouse look · `Shift` run · `M` map — (mobile: joystick + drag-to-look + RUN)

## Run it

Open `index.html` in any modern browser.
