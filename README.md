# Backrooms-game-
First-person Backrooms horror game: descend through 10 procedurally generated levels, each stalked by a different SCP anomaly (173, 096, 106, 682…). Manage sanity, stamina &amp; almond water, dodge the entities, find the exit. Single-file Three.js build — plays on PC &amp; mobile, installable as an offline PWA. SCP content CC BY-SA 3.0.
